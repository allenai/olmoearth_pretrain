# OlmoEarth scripts

This folder contains scripts run during the development of the OlmoEarth family of models.

Script directories are organized by stability:

- [`official`](official): supported launch/config scripts for released OlmoEarth model families. More documentation is available in [docs](../docs).
- [`vnext`](vnext): active experiments that may become future official recipes.
- [`tools`](tools): operational utilities for dataset analysis, cloud jobs, and one-off maintenance.
- [`base_loop_evals`](base_loop_evals): reusable evaluation-loop helpers.
- dated top-level directories: recent experiment-specific launch scripts that have not yet been archived.
- [`archived`](archived): historical experiment snapshots. These are kept for provenance and may not run against current versions of the codebase.
